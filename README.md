# <The Cult Scam> v2.0 (Crafting Edition)

A specialized guild utility for **The Cult**.

## Features
*   **[PLAYERS] Tab:** View all guild members and their professions. Click to expand.
*   **[SEARCH] Tab:** Search for any recipe (e.g., "Mongoose") to find crafters and required materials.
*   **[CHAT] Tab:** Private chat groups for officers, raiders, or scammers.
*   **Auto-Sync:** Opening your profession window automatically broadcasts your recipes to the guild.

## Usage
*   `/scam` - Toggle the window.
